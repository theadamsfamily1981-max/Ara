# ARA Cockpit Screenshots

This directory contains UI screenshots of the ARA Cockpit interface.

## Screenshots

| File | Description |
|------|-------------|
| `cockpit-storage.png` | Storage panel showing disk usage for snap packages |
| `cockpit-network-1.png` | Network panel view (home state) |
| `cockpit-network-2.png` | Network panel view (alternate state) |

## Adding Screenshots

Place PNG or JPG screenshots in this directory. Recommended naming convention:
- `cockpit-{panel-name}.png` - For panel-specific views
- `cockpit-{feature}-{description}.png` - For feature documentation

## Screenshot Guidelines

- **Resolution**: Match display resolution (typically 1920x1080 or native)
- **Format**: PNG preferred for UI screenshots
- **Content**: Capture the full cockpit window or specific panels
- **Naming**: Use lowercase with hyphens, descriptive names
